# Gushwork - Premium HDPE Pipes & Coils Landing Page

A modern, pixel-perfect responsive landing page for Gushwork, built with HTML, CSS, and JavaScript. This project showcases a premium HDPE pipes and coils manufacturer with a professional design and smooth interactions.

## 📁 Project Structure

```
gushwork-assignment/
├── index.html              # Main HTML file
├── css/
│   ├── styles.css         # Main stylesheet
│   └── responsive.css     # Responsive design rules
├── js/
│   └── script.js          # JavaScript functionality
└── assets/                # Image assets folder
```

## 🎨 Features

- **Responsive Design**: Mobile-first approach with breakpoints for tablet and desktop
- **Modern UI**: Clean, professional design with smooth animations
- **Interactive Elements**:
  - Expandable FAQ accordion
  - Smooth scroll navigation
  - Hover animations on cards and buttons
  - Number counter animations
  - Parallax effects
- **Performance Optimized**: 
  - Lazy loading for images
  - Smooth CSS transitions
  - Efficient JavaScript
- **Accessibility**: Semantic HTML, keyboard navigation support

## 🚀 Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- No build process required - works with plain HTML/CSS/JavaScript

### Setup

1. **Clone or download the project folder**
2. **Navigate to the project directory**
3. **Images Setup**: Add the following placeholder images to the `assets/` folder:
   - `hero-pipes.jpg` - Hero section main image (recommended size: 600x500px)
   - `app-1.jpg` - Water Distribution application image
   - `app-2.jpg` - Industrial Applications image
   - `app-3.jpg` - Irrigation Systems image
   - `app-4.jpg` - Gas Distribution image
   - `process-1.jpg` - Manufacturing process step 1
   - `process-2.jpg` - Manufacturing process step 2
   - `portfolio-1.jpg` - Portfolio item 1
   - `portfolio-2.jpg` - Portfolio item 2
   - `portfolio-3.jpg` - Portfolio item 3

4. **Open in browser**: Simply open `index.html` in your web browser

## 📝 Customization

### Change Colors
Edit the CSS variables in `css/styles.css`:
```css
:root {
    --primary-color: #1a3a7a;
    --secondary-color: #2c5aa0;
    --accent-color: #ff6b35;
    /* ... more colors ... */
}
```

### Update Content
Edit content directly in `index.html`:
- Update text in sections
- Change hero title
- Modify feature descriptions
- Update FAQ questions and answers

### Adjust Spacing
Modify spacing variables in `css/styles.css`:
```css
--spacing-xs: 4px;
--spacing-sm: 8px;
--spacing-md: 16px;
--spacing-lg: 24px;
--spacing-xl: 32px;
--spacing-2xl: 48px;
--spacing-3xl: 64px;
```
